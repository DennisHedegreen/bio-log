# bio-log

A simple public observation log of a large glass container with seawater.

The point is to observe slowly, document consistently, and only write larger articles when something actually changes or becomes interesting enough to analyze.

## Structure

- `logs/` daily observation notes
- `images/` photos grouped by date or event

## Suggested workflow

1. Observe the container
2. Take 1-3 photos
3. Write a short dated log entry
4. Only write a separate article when something unusual, meaningful, or analyzable happens

## First log format

Example:

```md
# 2026-03-21

- Water clarity:
- Visible organisms:
- Smell:
- Surface changes:
- Other notes:

Images:
- images/2026-03-21-01.jpg
- images/2026-03-21-02.jpg
```
